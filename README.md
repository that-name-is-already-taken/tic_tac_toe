# Tic Tac Toe

---

### Description

Digital version of the classic game built with Python and Pygame. Tic Tac Toe is a game where two players take turns
placing **X** and **O** on a 3x3 grid. The game detects a winner or a draw and then ends the round. There is no scoring
system - each game is played as a single round.

---

### Requirements

- Python 3.11
- Pygame

--- 

### How to run it?

1. Once you clone the repository, create virtual environment inside project and activate it

```
py -m venv .venv
.venv\Scripts\activate
```

2. Install Pygame

```
pip install pygame
```

3. Start the game with

```
python main.py
```
